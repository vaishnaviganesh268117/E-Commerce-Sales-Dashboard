# Ecommerce-Sales-Analytics-Dashboard
Interactive E-Commerce Sales Analytics Dashboard developed using Power BI.

## Project Overview

This project analyzes e-commerce sales data and presents business insights through an interactive Power BI dashboard. The workflow includes data storage using SQL, exploratory data analysis (EDA) using Python, and dashboard development using Power BI.

---

## Tools & Technologies

* SQL
* Python
* Pandas
* Sweetviz (Auto EDA)
* Power BI Desktop

---

## Project Workflow

### 1. Data Storage & Querying

The raw sales dataset was stored in a SQL database. SQL queries were used to retrieve, validate, and prepare the data for analysis.

### 2. Exploratory Data Analysis (EDA)

Python and Sweetviz were used to analyze the dataset, identify patterns, check data quality, and generate automated EDA reports.

### 3. Data Preparation

The dataset was cleaned and transformed to include key fields such as:

* Order Date
* Product Name
* Category
* Region
* Quantity
* Sales
* Profit
* Profit Margin
* Year & Month

### 4. Dashboard Development

The cleaned dataset was imported into Power BI to create an interactive four-page dashboard.

---

## Dashboard Pages

### Page 1 – E-Commerce Analytics

Provides an overall business overview.

**KPIs**

* Total Sales
* Total Profit
* Product Count
* Total Quantity Sold
* Profit Margin

**Visuals**

* Monthly Sales Trend
* Regional Sales Distribution
* Sales by Category
* Profit by Category

---

### Page 2 – Product Analysis

**Visuals**

* Top Products by Sales
* Top Products by Profit
* Product Sales Contribution
* Product Performance Analysis

**Purpose**

* Identify best-selling products
* Compare product profitability
* Analyze product contribution to revenue

---

### Page 3 – Regional Analysis

**Visuals**

* Sales by Region
* Profit by Region
* Quantity by Region
* Regional Performance Summary

**Purpose**

* Compare regional performance
* Identify high-performing regions
* Support regional business decisions

---

### Page 4 – Executive Insights & Recommendations

**Key Insights**

* Electronics generates the highest revenue and profit.
* West region records the highest sales and profit.
* Office category contributes the least revenue.
* Overall profit margin is approximately 17%.

**Recommendations**

* Focus on high-profit products.
* Improve sales in lower-performing regions.
* Increase inventory for top-selling categories.
* Monitor sales trends for forecasting.
